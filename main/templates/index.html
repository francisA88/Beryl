About Beryl


Beryl was created with the student in mind. It is intended to help students, especially at the tertiary level, by providing a way to easily search for solutions to past questions which may already exist out there on the Internet by simply supplying an image of a past question.


Beryl makes use of a powerful OCR (Optical Character Recognition) API to extract text from whatever image supplied. This extracted text is then searched on the Internet, focusing mainly on websites known to supply solved questions from experts like Chegg, Testbook, and more. Note that it can also be used to search for any text from any image aside from past questions.


A helpful tip - For best results, search using smaller sections of text rather than long texts.
